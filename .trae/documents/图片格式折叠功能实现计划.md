# 图片格式折叠功能实现计划

## 需求分析

### 功能描述
在文件夹中可能存在同一张图片的不同格式（如 DSC001.jpg 和 DSC001.raw），需要：
1. 在缩略图预览时折叠这些相同图片的不同格式
2. 优先以 jpg/png 格式作为缩略图显示目标
3. 在缩略图左上角添加一个约 30px 的半透明控件
4. 图标：&#xF156;
5. 鼠标移入移出事件控制 tip 显示

---

## 技术方案评估

### 问题1：图片加载逻辑评估

**官方优化方案（微软推荐）**

根据 [Microsoft Learn - Optimize file access for WinUI apps](https://learn.microsoft.com/da-dk/windows/apps/develop/performance/optimize-file-access)：

**核心优化技术：**

1. **使用系统索引器** - 最重要！
```csharp
var queryOptions = new QueryOptions();
queryOptions.IndexerOption = IndexerOption.UseIndexerWhenAvailable;
```
- 索引器缓存的属性可快速检索，避免磁盘 I/O
- 对于已索引的位置（如图片库），速度提升显著

2. **文件类型预过滤**
```csharp
var fileTypeFilter = new List<string> { ".jpg", ".jpeg", ".png", ".raw", ... };
var queryOptions = new QueryOptions(CommonFileQuery.OrderByName, fileTypeFilter);
```
- 在查询时就过滤，而不是加载后再过滤

3. **属性预取 (SetPropertyPrefetch)**
```csharp
queryOptions.SetPropertyPrefetch(
    PropertyPrefetchOptions.ImageProperties,
    new string[] { "System.Size" });
```
- 批量预取属性，避免逐个文件访问

4. **缩略图预取 (SetThumbnailPrefetch)**
```csharp
queryOptions.SetThumbnailPrefetch(
    ThumbnailMode.PicturesView,
    256,
    ThumbnailOptions.ReturnOnlyIfCached);
```

**方案A：先合并再加载（推荐，结合官方优化）**
```
使用索引器 + 文件类型过滤 → 快速扫描 → 建立分组 → 选择主图片 → 只加载主图片缩略图
```
- ✅ 使用索引器快速枚举文件
- ✅ 文件类型在查询时过滤
- ✅ 减少缩略图加载数量
- ✅ 避免UI闪烁
- ⚠️ 非索引位置会回退到磁盘扫描（但仍可用）

**方案B：加载时动态分组**
```
分批加载 → 动态分组 → 可能替换主图片
```
- ❌ 可能先显示 RAW 格式，然后切换到 JPG（闪烁）
- ❌ 加载了不必要的缩略图

**结论**：采用方案A + 官方优化API，理由：
1. `IndexerOption.UseIndexerWhenAvailable` 自动使用索引器加速
2. 文件类型预过滤减少不必要的文件对象创建
3. 减少不必要的缩略图加载（RAW 文件通常很大）
4. 避免UI闪烁，用户体验更好

### 问题2：文件扩展控件评估

**方案A：ToolTip（推荐用于纯展示）**
- ✅ 原生支持悬停显示
- ✅ 自动定位
- ✅ 无事件循环问题
- ❌ 无法在内容上停留（鼠标移出即关闭）
- ❌ 有默认延迟（可通过 InitialShowDelay="0" 解决）

**方案B：Flyout**
- ✅ 可自定义内容
- ✅ 支持复杂交互
- ❌ 默认需要点击触发
- ❌ PointerEntered/PointerExited 容易产生事件循环

**方案C：Popup**
- ✅ 完全可控
- ❌ 需要手动处理位置、显示/隐藏逻辑
- ❌ 代码复杂度高

**结论**：
- 如果只是显示格式信息：使用 **ToolTip**（简单稳定）
- 如果需要点击切换格式：使用 **Flyout** + 延迟关闭逻辑

**推荐方案**：使用 ToolTip，因为：
1. 用户需求是"移入显示，移出关闭"
2. ToolTip 最简单稳定，无事件循环问题
3. 通过 `ToolTipService.InitialShowDelay="0"` 实现秒显示

## 详细代码设计

### ImageFileInfo 新增属性

```csharp
// 已有属性
public ImageGroup? Group => _group;
public bool IsPrimary => _isPrimary;

// 需要新增
public List<ImageFileInfo>? AlternateFormats => 
    Group?.Images.Where(i => i != this).ToList();

public bool HasAlternateFormats => 
    AlternateFormats != null && AlternateFormats.Count > 0;

public string AlternateFormatsText
{
    get
    {
        if (!HasAlternateFormats || AlternateFormats == null)
            return string.Empty;
        
        var sb = new StringBuilder();
        sb.AppendLine($"当前: {FileType.ToUpperInvariant()}");
        sb.AppendLine();
        sb.AppendLine("其他格式:");
        foreach (var format in AlternateFormats)
        {
            sb.AppendLine($"• {format.FileType.ToUpperInvariant()}");
        }
        return sb.ToString().TrimEnd();
    }
}
```

### MainViewModel 加载逻辑修改（使用官方优化 API）

```csharp
public async Task LoadImagesAsync(FolderNode folderNode)
{
    _loadImagesCts?.Cancel();
    _loadImagesCts = new CancellationTokenSource();
    var cancellationToken = _loadImagesCts.Token;

    // ... 导航历史相关代码 ...

    Images.Clear();
    ImagesChanged?.Invoke(this, EventArgs.Empty);

    if (folderNode?.Folder == null)
        return;

    try
    {
        // 1. 创建优化的 QueryOptions
        var fileTypeFilter = ImageExtensions.ToList();
        var queryOptions = new QueryOptions(CommonFileQuery.OrderByName, fileTypeFilter)
        {
            IndexerOption = IndexerOption.UseIndexerWhenAvailable,
            FolderDepth = FolderDepth.Shallow
        };
        
        // 2. 预取图片属性（提升性能）
        queryOptions.SetPropertyPrefetch(
            PropertyPrefetchOptions.ImageProperties,
            new string[] { "System.Size", "System.DateModified" });

        var result = folderNode.Folder.CreateFileQueryWithOptions(queryOptions);
        
        // 3. 获取文件总数（用于进度显示）
        var totalCount = await result.GetItemCountAsync();
        
        // 4. 分批获取所有文件
        var allFiles = new List<StorageFile>();
        uint index = 0;
        const uint batchSize = 500; // 增大批次大小
        
        while (!cancellationToken.IsCancellationRequested)
        {
            var batch = await result.GetFilesAsync(index, batchSize);
            if (batch.Count == 0) break;
            allFiles.AddRange(batch);
            index += (uint)batch.Count;
            
            // 可选：报告进度
            // OnProgressChanged?.Invoke((double)allFiles.Count / totalCount);
        }
        
        // 5. 创建 ImageFileInfo 列表
        var imageInfos = await Task.Run(async () => {
            var tasks = allFiles.Select(f => LoadImageInfoSafeAsync(f, cancellationToken));
            var results = await Task.WhenAll(tasks);
            return results.Where(r => r != null).Cast<ImageFileInfo>().ToList();
        }, cancellationToken);
        
        // 6. 按文件名分组
        var groups = imageInfos
            .GroupBy(img => ImageGroup.GetGroupName(img.ImageFile.Name))
            .Select(g => new ImageGroup(g.Key, g))
            .ToList();
        
        // 7. 只添加主图片到 Images 集合
        foreach (var group in groups)
        {
            if (!cancellationToken.IsCancellationRequested)
            {
                group.PrimaryImage.UpdateDisplaySize(ThumbnailSize);
                Images.Add(group.PrimaryImage);
            }
        }

        ImagesChanged?.Invoke(this, EventArgs.Empty);
    }
    catch (OperationCanceledException) { }
    catch (Exception ex)
    {
        System.Diagnostics.Debug.WriteLine($"LoadImagesAsync error: {ex}");
    }
}

// 需要添加静态属性获取扩展名列表
private static List<string> ImageExtensionsList => ImageExtensions.ToList();
```

**性能优化要点：**
1. `IndexerOption.UseIndexerWhenAvailable` - 自动使用系统索引器
2. `SetPropertyPrefetch` - 批量预取属性，减少 I/O
3. 文件类型在查询时过滤，减少 StorageFile 对象创建
4. 增大批次大小 (500) 减少调用次数
```

## 注意事项

1. **线程安全**：所有 UI 对象必须在 UI 线程创建
2. **性能优化**：先分组再加载，减少不必要的缩略图加载
3. **用户体验**：ToolTip 使用 InitialShowDelay="0" 实现秒显示
4. **格式优先级**：确保所有支持的格式都有优先级定义
