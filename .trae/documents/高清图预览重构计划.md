# 高清图预览重构计划 - 动画层 + 查看层分离架构

## 当前问题

当前实现过于复杂，存在以下问题：
1. **一个 ScrollViewer 内不断换图 + 调 zoom + 调 offset** - 最难路线
2. **动画过程中换图** - 最难
3. **动画过程中缩放** - 最难
4. **动画过程中坐标同步** - 最难
5. **每张图继承上一张图的缩放** - 不需要

## 重构目标

采用 **"动画层+ 查看层分离架构"**，类似 Win11 照片查看器：
- 动画阶段：只用固定布局的 Image
- 查看阶段：使用 ScrollViewer + Image

---

## 核心思路

### 阶段1：动画阶段（简单）
- 只做 ConnectedAnimation：缩略图 → 居中 Image（固定布局）
- ❌ 不要 ScrollViewer
- ❌ 不要 zoom
- ❌ 不要渐进加载
- ✅ 只用 `<Image Stretch="Uniform"/>`
- ✅ 保证动画绝对稳定

### 阶段2：动画结束后（简单）
- 静态 Image → ScrollViewer + Image
- 再开始加载高清图
- 允许缩放拖拽
- 每张图独立计算 fitScale，不继承上一张图的缩放

---

## UI 结构

```xml
<Grid>
    <!-- 阶段1：动画用 -->
    <Image x:Name="AnimationImage"
           Stretch="Uniform"
           HorizontalAlignment="Center"
           VerticalAlignment="Center"
           Opacity="1"/>

    <!-- 阶段2：查看器 -->
    <ScrollView x:Name="ImageScrollView"
                ZoomMode="Enabled"
                MinZoomFactor="0.1"
                MaxZoomFactor="10.0"
                Opacity="0">
        <Grid HorizontalAlignment="Stretch" VerticalAlignment="Stretch">
            <Image x:Name="MainImage"
                   Stretch="None"
                   HorizontalAlignment="Center"
                   VerticalAlignment="Center"/>
        </Grid>
    </ScrollView>
</Grid>
```

---

## 流程设计

### 1️⃣ 开始动画
```
缩略图 → AnimationImage
启动 ConnectedAnimation
```

### 2️⃣ 动画结束
```
AnimationImage.Opacity = 0
ImageScrollView.Opacity = 1
```

### 3️⃣ 初始化 viewer
```
设置 MainImage.Source = 缩略图
计算 fitScale（基于当前图片尺寸）
应用 fitScale
```

### 4️⃣ 异步加载高清图
```
直接替换 MainImage.Source
重新计算 fitScale
应用新 fitScale
```

---

## 关键优势

| 对比项 | 当前方案 | 新方案 |
|-------|---------|--------|
| 动画过程 | 换图 + 缩放 + 坐标同步 | 只做动画，固定布局 |
| 高清图加载 | 复杂的 zoom 计算 | 直接替换，重新计算 fitScale |
| 缩放继承 | 继承上一张图 | 每张图独立计算 |
| 复杂度 | 最难 | 最简单 |

## 修改步骤

### 步骤 1：修改 XAML 结构
- [ ] 添加 `AnimationImage` 用于动画
- [ ] 保留 `ImageScrollView` 和 `MainImage` 用于查看
- [ ] 初始状态：AnimationImage.Opacity=1, ImageScrollView.Opacity=0

### 步骤 2：修改 PrepareContent
- [ ] 只设置 AnimationImage.Source = 缩略图
- [ ] 不设置 MainImage.Source

### 步骤 3：修改 PrepareForAnimationAsync
- [ ] 准备 ConnectedAnimation（使用 AnimationImage）
- [ ] 不需要等待布局完成
- [ ] 不需要计算 fitScale

### 步骤 4：修改 ShowAfterAnimationAsync
- [ ] 动画结束后：
  1. AnimationImage.Opacity = 0
  2. ImageScrollView.Opacity = 1
  3. 设置 MainImage.Source = 缩略图
  4. 计算 fitScale 并应用
  5. 启动高清图加载

### 步骤 5：简化 LoadHighResolutionImageAsync
- [ ] 直接替换 MainImage.Source
- [ ] 重新计算 fitScale
- [ ] 不需要锁定尺寸、坐标换算等复杂逻辑

### 步骤 6：修改 PrepareCloseAnimation
- [ ] 准备返回动画（使用 AnimationImage）

- ✅ 高清图加载简单（直接替换，重新计算 fitScale）
- ✅ 每张图独立（不继承上一张图的缩放）
- ✅ 无跳变、无闪烁、无变形
