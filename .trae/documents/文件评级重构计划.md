# 文件评级功能重构计划

## 目标
重构文件评级读写功能，使用 WinRT API 实现异步评级读写，并在缩略图下方添加评级交互控件。

## 需求概述
1. 使用 `Windows.Storage.StorageFile.Properties.GetImagePropertiesAsync()` 和 `ImageProperties.Rating` 进行读写
2. 缩略图下方添加 180×35 半透明评级框，内含 RatingControl 控件
3. 评级框常驻显示（仅对支持的文件类型），未评级显示空星
4. 异步读写时使用 Shimmer 控件展示加载骨架图
5. 支持的文件格式：JPG/JPEG、TIFF/TIF、PNG（部分）、HEIC/HEIF（Win11+）、DNG

### 步骤 2：创建 RatingService 服务
**新文件**: `Services/RatingService.cs`

职责：
- 提供异步读取文件评级的方法 `GetRatingAsync(StorageFile file)`
- 提供异步写入文件评级的方法 `SetRatingAsync(StorageFile file, uint rating)`
- 提供文件类型支持检测方法 `IsRatingSupported(string fileExtension)`
- 使用 `SemaphoreSlim` 限制并发（4-8个）
- 正确处理异常（不支持评级的文件、文件被占用等）

支持的文件扩展名：
```csharp
private static readonly HashSet<string> SupportedExtensions = new(StringComparer.OrdinalIgnoreCase)
{
    ".jpg", ".jpeg", ".tif", ".tiff", ".png", ".heic", ".heif", ".dng"
};
```

Rating 值范围：0-99（WinRT API 标准）
- 0 = 未评级
- 1-25 = 1 星
- 26-50 = 2 星
- 51-75 = 3 星
- 76-99 = 4 星（部分系统 76-99 为 5 星）

   ```csharp
   {
       get => _isRatingLoading;
       set => SetProperty(ref _isRatingLoading, value);
   }

   {
       get => _isRatingSupported;
       private set => SetProperty(ref _isRatingSupported, value);
   }
   ```
3. **移除** `ImageRating` 只读属性
4. **添加** 异步加载评级方法：
   ```csharp
   public async Task LoadRatingAsync(RatingService ratingService)
   {
       IsRatingLoading = true;
       IsRatingSupported = ratingService.IsRatingSupported(ImageFile.FileType);
       
       if (!IsRatingSupported)
       {
           IsRatingLoading = false;
           return;
       }
       
       try
       {
           Rating = await ratingService.GetRatingAsync(ImageFile);
       }
       finally
       {
           IsRatingLoading = false;
       }
   }
   ```
5. **添加** 异步设置评级方法：
   ```csharp
   public async Task SetRatingAsync(RatingService ratingService, int newRating)
   {
       if (!IsRatingSupported) return;
       
       IsRatingLoading = true;
       try
       {
           await ratingService.SetRatingAsync(ImageFile, (uint)newRating);
           Rating = newRating;
       }
       finally
       {
           IsRatingLoading = false;
       }
   }
   ```

### 步骤 6：添加必要的转换器
**新文件**: `Helpers/RatingConverter.cs`

将 WinRT Rating 值（0-99）转换为 RatingControl 的星级值（0-5）：
```csharp
public class RatingConverter : IValueConverter
{
    public object Convert(object value, Type targetType, object parameter, string language)
    {
        if (value is int rating)
        {
            // 0-99 转换为 0-5 星
            if (rating == 0) return 0;
            if (rating <= 25) return 1;
            if (rating <= 50) return 2;
            if (rating <= 75) return 3;
            return 4; // 或 5，取决于系统定义
        }
        return 0;
    }
    
    public object ConvertBack(object value, Type targetType, object parameter, string language)
    {
        if (value is int stars)
        {
            // 0-5 星转换为 0-99
            return stars switch
            {
                0 => 0,
                1 => 25,
                2 => 50,
                3 => 75,
                4 => 99,
                _ => 0
            };
        }
        return 0;
    }
}
```

```csharp
private async void RatingControl_ValueChanged(RatingControl sender, object args)
{
    if (sender.DataContext is ImageFileInfo imageInfo)
    {
        await imageInfo.SetRatingAsync(_ratingService, (int)sender.Value);
    }
}
```
