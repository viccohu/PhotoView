| 需求 | 说明 |

## 技术要点

### 路径存储格式

存储完整路径字符串：
```
C:\Users\XXX\Pictures
D:\Photos
```

### 路径恢复流程

```
1. 应用启动 → ShellPage 加载所有设置
2. MainPage Loaded → 检查 RememberLastFolder
3. 如果 true → 尝试获取 LastFolderPath
4. 遍历 FolderTree 查找匹配路径的节点
5. 如果找到 → 调用 ExpandTreeViewPathAsync 展开并加载
6. 如果没找到 → 显示"这台电脑"
```
