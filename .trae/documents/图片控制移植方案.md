# 图片控制移植方案

## 参考项目分析

参考项目 (FastPick) 实现了一套成熟的图片控制系统，包含以下核心特性：

### 1. XAML 结构

```xml
<Grid x:Name="PreviewContainer" 
      PointerWheelChanged="PreviewContainer_PointerWheelChanged"
      PointerPressed="PreviewContainer_PointerPressed"
      PointerMoved="PreviewContainer_PointerMoved"
      PointerReleased="PreviewContainer_PointerReleased"
      DoubleTapped="PreviewContainer_DoubleTapped">
    <Grid.Clip>
        <RectangleGeometry x:Name="PreviewClip" />
    </Grid.Clip>
    
    <Grid x:Name="PreviewImageContainer"
          HorizontalAlignment="Center"
          VerticalAlignment="Center"
          RenderTransformOrigin="0.5,0.5">
        <Grid.RenderTransform>
            <TransformGroup>
                <ScaleTransform x:Name="PreviewScaleTransform" ScaleX="1" ScaleY="1" />
                <RotateTransform x:Name="PreviewRotateTransform" Angle="0" />
                <TranslateTransform x:Name="PreviewTranslateTransform" X="0" Y="0" />
            </TransformGroup>
        </Grid.RenderTransform>
        <Image x:Name="PreviewImageFront" Stretch="Uniform" ... />
    </Grid>
</Grid>
```

**关键设计**：

* 使用 `Grid.Clip` + `RectangleGeometry` 实现图片裁剪，让图片感知容器边界

* 使用 `TransformGroup` 分离缩放、旋转、平移变换

* `RenderTransformOrigin="0.5,0.5"` 确保变换以中心为基准

### 2. 核心状态变量

```csharp
private double _zoomScale = 1.0;           // 当前缩放比例（相对于 Fit 状态）
private double _minZoom = 0.1;             // 最小缩放

private const double SpringBackFactor = 0.12;    // 回弹因子
private const double VelocityThreshold = 0.5;    // 速度阈值

// 缩放锚点（鼠标中心缩放）

private double _zoomAnchorImgY = 0;
```

### 3. 物理驱动引擎

参考项目使用 `CompositionTarget.Rendering` 事件驱动物理模拟：

```csharp
// 订阅渲染帧事件
Microsoft.UI.Xaml.Media.CompositionTarget.Rendering += OnPhysicsRendering;

private void OnPhysicsRendering(object sender, object e)
{
    bool needsUpdate = false;
    bool isZooming = Math.Abs(_zoomScale - _targetZoomScale) > 0.0001;
    
    // 1. 缩放缓动
    if (isZooming)
    {
        _zoomScale += (_targetZoomScale - _zoomScale) * ZoomEasingFactor;
        // 鼠标中心缩放逻辑...
        ClampTranslation();
        needsUpdate = true;
    }
    
    // 2. 惯性移动
    if (!isZooming && !_isDragging && (Math.Abs(_velocityX) > VelocityThreshold || ...))
    {
        _translateX += _velocityX;
        _translateY += _velocityY;
        _velocityX *= InertiaDamping;
        _velocityY *= InertiaDamping;
        needsUpdate = true;
    }
    
    // 3. 弹性边界回弹
    if (!isZooming)
    {
        needsUpdate |= ApplyBoundsWithDamping();
    }
    
    if (needsUpdate)
    {
        ApplyZoomTransform();
    }
}
```

### 4. 边界控制

**硬边界限制**（缩放时）：

```csharp
private void ClampTranslation()
{
    var (imageWidth, imageHeight) = GetRotatedImageSize();
    
    // 图片小于容器时，强制居中
    if (imageWidth <= containerWidth)
        _translateX = 0;
    else
        _translateX = Math.Clamp(_translateX, -maxTranslateX, maxTranslateX);
    
    // 垂直方向同理...
}
```

**弹性边界**（惯性/拖动后）：

```csharp
private bool ApplyBoundsWithDamping()
{
    // 图片小于容器时，回弹到居中
    if (imageWidth <= containerWidth)
    {
        if (Math.Abs(_translateX) > 0.5)
        {
            _translateX *= (1 - SpringBackFactor);
            _velocityX = 0;
            return true;
        }
    }
    else
    {
        // 超出边界时，阻尼回弹
        if (_translateX > maxTranslateX)
        {
            var over = _translateX - maxTranslateX;
            _translateX = maxTranslateX + over * BoundaryDamping;
            _velocityX *= -BoundaryDamping;
            return true;
        }
    }
}
```

### 5. 缩放控制

**鼠标中心缩放**：

```csharp
private void PreviewContainer_PointerWheelChanged(...)

    _zoomAnchorImgY = imgY;
    _zoomAnchorScreenX = mouse.X;
    _zoomAnchorScreenY = mouse.Y;
    _hasZoomAnchor = true;
    
    // 应用缩放
    double scaleFactor = Math.Pow(1.0015, delta);

```

**自适应缩放计算**：

```csharp
// 计算图片适应容器的缩放比例
private double CalculateFitToScreenScale(PhotoItem item)
{
    var scaleX = containerWidth / item.Width;
    var scaleY = containerHeight / item.Height;
    return Math.Min(scaleX, scaleY);
}

// 计算原图 100% 显示对应的 _zoomScale
private double CalculateOriginalScale(PhotoItem item)
{
    var fitScale = CalculateFitToScreenScale(item);
    var dpiScale = GetDpiScale();
    return (1.0 / fitScale) / dpiScale;
}
```

### 6. 拖动控制

```csharp
private void PreviewContainer_PointerMoved(...)
{
    if (_isDragging)
    {
        var deltaX = currentPoint.X - _lastDragPoint.X;
        var deltaY = currentPoint.Y - _lastDragPoint.Y;
        
        // 检查是否允许拖动
        if (CanPanHorizontal())
        {
            _translateX += deltaX;
            _velocityX = deltaX;  // 记录速度
        }
        
        if (CanPanVertical())
        {
            _translateY += deltaY;
            _velocityY = deltaY;
        }
    }
}
```

## 移植方案

### 需要移植的功能

1. **图片边界控制** - `ClampTranslation()` + `ApplyBoundsWithDamping()`
2. **缩放控制** - 鼠标中心缩放 + 100% 吸附
3. **拖动控制** - 带惯性的拖动
4. **物理效果** - 惯性衰减 + 弹性回弹
5. **容器裁剪** - 让图片感知容器边界

### 不需要移植的功能

* 分级加载（已有实现）

* 双缓冲预览

* 智能预加载

* 浏览方向检测

## 关键差异对比

| 特性     | 当前实现               | 参考实现                                  |
| ------ | ------------------ | ------------------------------------- |
| 变换方式   | CompositeTransform | TransformGroup (Scale + Translate 分离) |
| 边界控制   | 硬限制                | 硬限制 + 弹性回弹                            |
| 缩放动画   | 无缓动                | 缓动过渡                                  |
| 惯性效果   | 无                  | 有                                     |
| 容器裁剪   | 无                  | RectangleGeometry.Clip                |
| 鼠标中心缩放 | 简单实现               | 完整锚点系统                                |

***

## 注意事项

1. **线程安全** - 物理引擎在渲染线程运行，需要确保 UI 操作在 UI 线程
2. **性能** - 每帧都会调用物理引擎，需要优化计算
3. **DPI 缩放** - 需要考虑 DPI 缩放比例
4. **旋转支持** - 当前项目暂不支持旋转，可简化相关逻辑
