# 窗口级快捷键处理方案

## 问题分析

### 当前状况
- **MainPage** 和 **CollectPage** 各自处理快捷键
- 快捷键依赖页面元素的焦点状态
- 焦点丢失后快捷键失效

### 共同快捷键
| 快捷键 | MainPage | CollectPage |
|--------|----------|-------------|
| Delete | 切换删除标记 | 切换删除标记 |
| 0-5 | 设置评分 | 设置评分 |
| Left/Right/Up/Down | 切换图片（查看器）| 移动选择 |
| Escape | 清除选择/关闭查看器 | 重置缩放 |
| Space | 切换图片查看器 | - |
| Ctrl+A | 全选 | - |

## 解决方案

### 架构设计

```
App (Window)
  └─ ShellPage
      ├─ MainPage (处理特定快捷键)
      └─ CollectPage (处理特定快捷键)
```

## 实施计划

### 第一步：创建快捷键服务接口和实现

#### 1.1 创建 IKeyboardShortcutService 接口
```csharp
namespace PhotoView.Contracts.Services;

public interface IKeyboardShortcutService
{
    void Initialize(Window window);
    void RegisterPageShortcutHandler(string pageKey, Func<KeyRoutedEventArgs, bool> handler);
    void UnregisterPageShortcutHandler(string pageKey);
    void SetCurrentPage(string pageKey);
    bool IsTextInputFocused();
}
```

#### 1.2 创建 KeyboardShortcutService 实现
```csharp
namespace PhotoView.Services;

public class KeyboardShortcutService : IKeyboardShortcutService
{
    private Window? _window;
    private readonly Dictionary<string, Func<KeyRoutedEventArgs, bool>> _pageHandlers = new();
    private string _currentPageKey = "";

    public void Initialize(Window window)
    {
        _window = window;
        window.DispatcherQueue.TryEnqueue(() =>
        {
            // 在窗口级别注册键盘事件
            if (window.Content is UIElement rootElement)
            {
                rootElement.KeyDown += Window_KeyDown;
            }
        });
    }

    private void Window_KeyDown(object sender, KeyRoutedEventArgs e)
    {
        // 排除输入框
        if (IsTextInputFocused())
            return;

        // 调用当前页面的处理器
        if (_pageHandlers.TryGetValue(_currentPageKey, out var handler))
        {
            if (handler(e))
            {
                e.Handled = true;
            }
        }
    }

    public bool IsTextInputFocused()
    {
        if (_window?.Content is not UIElement rootElement)
            return false;

        var focusedElement = FocusManager.GetFocusedElement(rootElement.XamlRoot) as DependencyObject;
        return IsTextInputElement(focusedElement);
    }

    private static bool IsTextInputElement(DependencyObject? element)
    {
        while (element != null)
        {
            if (element is TextBox or PasswordBox or RichEditBox or AutoSuggestBox)
                return true;

            element = VisualTreeHelper.GetParent(element);
        }
        return false;
    }
}
```

### 第三步：创建基础快捷键处理器

#### 3.1 创建 BaseShortcutHandler 基类
```csharp
namespace PhotoView.Services;

public abstract class BaseShortcutHandler
{
    protected readonly IKeyboardShortcutService _shortcutService;
    protected readonly ISettingsService _settingsService;

    protected BaseShortcutHandler(
        IKeyboardShortcutService shortcutService,
        ISettingsService settingsService)
    {
        _shortcutService = shortcutService;
        _settingsService = settingsService;
    }

    protected bool HandleCommonShortcuts(KeyRoutedEventArgs e, 
        Action? onDelete = null,
        Action<uint>? onRating = null,
        Action<int>? onNavigate = null,
        Action? onEscape = null)
    {
        switch (e.Key)
        {
            case VirtualKey.Delete:
                onDelete?.Invoke();
                return true;

            case VirtualKey.Escape:
                onEscape?.Invoke();
                return true;

            case >= VirtualKey.Number0 and <= VirtualKey.Number5:
                onRating?.Invoke((uint)(e.Key - VirtualKey.Number0));
                return true;

            case >= VirtualKey.NumberPad0 and <= VirtualKey.NumberPad5:
                onRating?.Invoke((uint)(e.Key - VirtualKey.NumberPad0));
                return true;

            case VirtualKey.Left:
            case VirtualKey.Up:
                onNavigate?.Invoke(-1);
                return true;

            case VirtualKey.Right:
            case VirtualKey.Down:
                onNavigate?.Invoke(1);
                return true;
        }

        return false;
    }
}
```

### 第四步：修改 MainPage 使用新服务

#### 4.1 在 MainPage 构造函数中注册
```csharp
public MainPage(
    MainViewModel viewModel,
    IKeyboardShortcutService shortcutService,
    ISettingsService settingsService)
{
    InitializeComponent();
    ViewModel = viewModel;
    
    // 注册快捷键处理器
    shortcutService.RegisterPageShortcutHandler("MainPage", HandleShortcut);
}

protected override void OnNavigatedTo(NavigationEventArgs e)
{
    base.OnNavigatedTo(e);
    var shortcutService = App.GetService<IKeyboardShortcutService>();
    shortcutService.SetCurrentPage("MainPage");
}

private bool HandleShortcut(KeyRoutedEventArgs e)
{
    // 处理 MainPage 特定的快捷键
    if (_currentViewer != null)
    {
        return HandleViewerShortcut(e);
    }
    
    return HandleMainPageShortcut(e);
}
```

## 优势

### 1. 统一管理
- 所有快捷键在一个地方管理
- 避免重复代码
- 易于维护和扩展

### 2. 焦点无关
- 窗口级事件处理
- 不依赖页面元素焦点
- 解决焦点丢失问题

### 3. 灵活扩展
- 支持多页面
- 支持动态注册/注销
- 易于添加新快捷键

### 4. 性能优化
- 单一事件监听
- 减少事件处理器数量
- 提高响应速度

## 风险评估

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 窗口级事件可能影响其他控件 | 中 | 精确检测输入框焦点 |
| 页面切换时处理器未注销 | 低 | 在 OnNavigatedFrom 中注销 |
| 快捷键冲突 | 低 | 明确定义优先级规则 |
