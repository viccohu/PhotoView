# 导出功能实现方案

## 需求概述

添加导出功能，支持按星级筛选导出图片和 RAW 文件。

---

## 功能设计

### 1. 导出按钮和 Flyout

**位置**：右上角功能区（删除按钮旁边）

**Flyout 布局**：
```
┌─────────────────────────────────────────────────────────────┐
│  [ ] 图片  [≥ 1-5星▼]  [文件夹名称: ______]  [ ] RAW  [≥ 1-5星▼]  [文件夹名称: ______] │
│                                                             │
│  导出地址: _________________________  [📁 选择文件夹]       │
│  [────────────────────────────────────] (进度条)            │
│  [取消]  [开始导出]                                           │
└─────────────────────────────────────────────────────────────┘
```

### 2. 控件说明

| 控件 | 类型 | 默认值 | 说明 |
|-----|------|--------|-----|
| 图片 | ToggleButton | 禁用 | 启用后激活图片相关控件 |
| RAW | ToggleButton | 禁用 | 启用后激活 RAW 相关控件 |
| 星级选择 | ComboBox | ≥ 1 星 | 选项：≥ 1-5 星 |
| 文件夹名称 | TextBox | 图片为空、RAW为"RAW" | 为空或"/"、"\"时不创建子文件夹 |
| 导出地址 | TextBox | 记住上次路径 | |
| 选择文件夹 | Button | | 打开文件夹选择器 |
| 进度条 | ProgressBar | 0 | 显示导出进度 |
| 取消 | Button | | 取消导出 |
| 开始导出 | Button | | 开始导出 |

### 3. 设置持久化

添加以下设置项：
- `ExportLastFolderPath` - 上次导出地址
- `ExportImageEnabled` - 图片导出是否启用
- `ExportImageMinRating` - 图片最小星级
- `ExportImageFolderName` - 图片子文件夹名称
- `ExportRawEnabled` - RAW 导出是否启用
- `ExportRawMinRating` - RAW 最小星级
- `ExportRawFolderName` - RAW 子文件夹名称

---

## 文件复制逻辑

```
导出流程：
1. 收集所有图片（预删除的不导出）
2. 按筛选条件过滤
3. 创建目标文件夹（如果需要）
4. 逐个复制文件
5. 更新进度条
```
