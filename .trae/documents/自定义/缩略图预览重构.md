# 一、目标效果（明确你要做的不是普通瀑布流）

Win11照片布局本质是：

### ✅ 特点

- 每一行 **高度一致（固定行高）**
- 每一行图片 **宽度自适应（按比例伸缩）**
- 行宽 **刚好填满容器（无空隙）**
- 图片不会被裁剪（保持比例）
- 行数根据容器宽度动态变化

👉 这不是 WrapGrid / UniformGrid\
👉 也不是瀑布流（Pinterest）\
👉 本质是：**Justified Layout（两端对齐布局）**

***

# 三、自定义 Layout 设计（核心）

## ✅ 1. 定义布局类

```
public class JustifiedLayout : Layout
{
    public double LineHeight { get; set; } = 200;
    public double Spacing { get; set; } = 4;

    protected override Size MeasureOverride(LayoutContext context, Size availableSize)
    {
        // 核心：计算每一行怎么分
    }

    protected override Size ArrangeOverride(LayoutContext context, Size finalSize)
    {
        // 核心：摆放元素
    }
}
```

***

## ✅ 2. 核心算法（重点讲清楚）

### 👉 输入

- &#x20;容器宽度：`containerWidth`&#x20;
- &#x20;每张图片比例：`aspectRatio = width / height`&#x20;
- &#x20;目标行高：`LineHeight`&#x20;

***

## 👉 步骤（Win11核心逻辑）

### Step 1：累加图片，直到填满一行

```
sum(width) + spacing ≈ containerWidth
```

但宽度是：

```
width = aspectRatio * lineHeight
```

***

### Step 2：如果不满 → 拉伸整行

计算缩放比例：

```
scale = containerWidth / currentRowWidth
```

然后：

```
finalHeight = lineHeight * scale
finalWidth = width * scale
```

👉 这一步就是“等高 + 填满”

***

## ✅ 3. 行数据结构（建议）

```
class Row
{
    public List<int> Indices;
    public double Height;
}
```

***

# 四、MeasureOverride 实现思路

```
protected override Size MeasureOverride(LayoutContext context, Size availableSize)
{
    double width = availableSize.Width;

    var rows = BuildRows(context, width);

    double totalHeight = 0;

    foreach (var row in rows)
    {
        foreach (var index in row.Indices)
        {
            var element = context.GetOrCreateElementAt(index);
            element.Measure(new Size(double.PositiveInfinity, double.PositiveInfinity));
        }

        totalHeight += row.Height + Spacing;
    }

    return new Size(width, totalHeight);
}
```

***

# 五、ArrangeOverride 实现思路

```
protected override Size ArrangeOverride(LayoutContext context, Size finalSize)
{
    double y = 0;

    foreach (var row in rows)
    {
        double x = 0;

        foreach (var index in row.Indices)
        {
            var element = context.GetOrCreateElementAt(index);

            double width = row.Height * aspectRatio;

            element.Arrange(new Rect(x, y, width, row.Height));

            x += width + Spacing;
        }

        y += row.Height + Spacing;
    }

    return finalSize;
}
```

# 六、关键优化点（非常重要）

## ✅ 1. 不要每次都重新算 rows

👉 必须缓存：

```
List<Row> _rowsCache;
double _lastWidth;
```

```
if (_lastWidth != availableSize.Width)
{
    _rowsCache = BuildRows(...);
    _lastWidth = availableSize.Width;
}
```

***
