# 🧠 🎯 PhotoView 优化指令（AI执行版）

## 🔴 任务目标

```
重构当前 WinUI 3 图片浏览器，解决以下问题：
1. 快速滚动崩溃（0xc000027b）
2. 图片错位 / 闪烁
3. 滚动卡顿
4. 大文件夹加载性能差

要求：
- 使用 ItemsRepeater + 虚拟化
- 完全数据驱动 UI
- 保证线程安全
```

***

# 🔥 一、必须执行的修改（阻止崩溃）

## 1️⃣ 修改 ImageFileInfo（线程安全）

```
修改 GetThumbnailAsync：
- 所有 BitmapImage 创建必须在 UI线程执行
- 使用 DispatcherQueue 包裹 SetSource
```

```
await App.MainWindow.DispatcherQueue.EnqueueAsync(() =>
{
    bitmap = new BitmapImage();
    bitmap.SetSource(stream);
});
```

***

## 2️⃣ 修改 EnsureThumbnailAsync（防复用）

```
添加版本控制字段 _loadVersion
确保 async 返回时不会写入已复用对象
```

```
private int _loadVersion;

var version = ++_loadVersion;

var result = await GetThumbnailAsync(size);

if (version != _loadVersion)
    return;

Thumbnail = result;
```

***

## 3️⃣ 删除 UI 层图片操作

```
删除所有以下代码：
- thumbnailImage.Source =
- GetImageAndIndicatorFromContentControl
- UpdateThumbnails()
```

***

## 4️⃣ 修改 XAML（绑定驱动）

<ProgressRing IsActive="{x:Bind Thumbnail == null}" />
```

## 5️⃣ 修改 ElementPrepared

只触发加载，不操作 UI

```

# 🚀 二、性能优化（必须）

## 6️⃣ 修改 LoadImagesAsync（后台加载）

```
将图片信息加载移到后台线程
避免 UI线程 await 循环
```

```
var list = await Task.Run(async () =>
{
    var temp = new List<ImageFileInfo>();
    foreach (var file in imageFiles)
    {
        if (IsImageFile(file))
            temp.Add(await LoadImageInfo(file));
    }
    return temp;
});
```

# 🚨 六、验收标准（AI必须满足）
