&#x20;

## 一、整体布局结构

### 1. 根容器

- **控件类型**：`ScrollViewer`（垂直滚动，适配内容超出屏幕场景）
- **布局容器**：内部嵌套 `VerticalStackPanel`（垂直方向流式布局，统一间距）
- **间距配置**：`Spacing="16"`（模块间垂直间距）
- **边距配置**：`Padding="16,12,16,12"`（整体内边距）
- **主题适配**：默认深色主题，支持浅色主题自动适配

### 2. 模块划分（从上到下）

1. **文件名模块**（图标 + 文件名输入 / 展示）
2. **评级模块**（图标 + 星级评分控件）
3. **拍摄日期模块**（图标 + 日期时间选择器）
4. **文件信息模块**（标题 + 2 行 ×3 列 固定大小网格）
5. **设备信息模块**（标题 + 2 行 ×3 列 固定大小网格）

## 三、各模块详细布局与样式规范

### 1. 文件名模块

#### 布局结构

- **容器**：`HorizontalStackPanel`（水平布局，图标 + 内容）
- **间距**：`Spacing="12"`
- **边距**：`Margin="0,0,0,8"`
- **控件组成**：
  1. `FontIcon`：图标 `&#xE91B;`，`FontSize="16"`，`VerticalAlignment="Center"`
  2. `TextBlock`（展示态）/ `TextBox`（编辑态）：
     - 内容：`DSC_0057`（示例）
     - 样式：`FontSize="18"`，`FontWeight="SemiBold"`，`VerticalAlignment="Center"`
     - 最大宽度：自适应，超出显示省略号

#### XAML 示例

xml

```
<HorizontalStackPanel Spacing="12" Margin="0,0,0,8">
    <FontIcon FontFamily="Segoe MDL2 Assets" Glyph="&#xE91B;" FontSize="16" VerticalAlignment="Center" />
    <TextBlock Text="DSC_0057" FontSize="18" FontWeight="SemiBold" VerticalAlignment="Center" TextTrimming="CharacterEllipsis" MaxWidth="300" />
</HorizontalStackPanel>

```

## 四、全局样式与主题适配

### 1. 深色主题默认样式（核心）

- 背景色：根容器 `Background="#1E1E1E"`（WinUI3 深色主题默认背景）
- 格子背景：`#2B2B2B`（比根背景浅一档，符合 Fluent Design 层级）
- 文字颜色：默认 `White`，标签文字 `Opacity="0.7"`（弱化层级）
- 圆角：所有交互控件 / 格子统一 `CornerRadius="4"`（小圆角）
- 间距：严格遵循模块间 `16px`、格子间 `8px`、内容间 `4px/12px`

### 2. 浅色主题适配

- 根背景：`White`
- 格子背景：`#F5F5F5`
- 文字颜色：默认 `Black`，标签文字 `Opacity="0.8"`
- 自动适配：通过 WinUI3 `ThemeResource` 实现，无需手动写两套样式

### 3. 性能优化要点

- 网格使用 `UniformGrid` 而非 `Grid`，自动均分固定大小，减少布局计算
- 图标统一使用 `FontIcon`，避免图片资源，提升渲染性能
- 根容器用 `ScrollViewer` 包裹，适配不同屏幕高度，避免内容溢出

## 六、AI 开发工具实现提示（给 Cursor/Trae 用）

1. **严格遵循 WinUI3 官方控件**：所有控件必须使用 `Microsoft.UI.Xaml.Controls` 命名空间下的官方控件，禁止第三方控件
2. **主题资源替换**：将硬编码背景色 `#2B2B2B` 替换为 `{ThemeResource SubtleFillColorSecondaryBrush}`，自动适配深浅主题
3. **数据绑定**：所有展示文本（文件名、尺寸、相机参数等）需预留绑定路径，例如 `{x:Bind FileName}`，方便后续对接业务逻辑
4. **性能优化**：`UniformGrid` 是 WinUI3 中实现固定大小网格的最优选择，避免手动设置 `Grid` 列宽行高
5. **交互逻辑**：`RatingControl`、`CalendarDatePicker`、`TimePicker` 需添加 `ValueChanged` 事件，实现数据回写
