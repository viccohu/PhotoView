# 主页标题布局优化计划

## 目标

修改主页标题区域的布局，使其显示：
1. 第一行：[文件夹名称][功能区]（文字与按钮横向高度对齐，功能区按钮靠右）
2. 第二行：[图片数量]（灰色小字）
3. 工具图标统一移动至标题栏功能区
4. 整个标题区域维持原本布局位置

## 当前布局分析

### 当前标题区域（MainPage.xaml:63-107）
- 使用 Grid 布局，两列
- 第一列：BreadcrumbBar（面包屑导航）
- 第二列：StackPanel（工具按钮：缩略图大小、编辑按钮）

### 需要修改的部分
1. 移除 BreadcrumbBar
2. 添加文件夹名称显示
3. 添加图片数量显示
4. 重新组织布局为两行
