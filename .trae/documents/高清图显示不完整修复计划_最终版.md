# 高清图显示不完整问题修复计划（最终版）

## 问题分析

### 核心问题

1. **高清图尺寸获取失败**（关键）
   - 无法获取高清图的 `PixelWidth/PixelHeight`
   - 导致使用了缩略图的缩放比例（2.578）
   - 结果：高清图被放大到 15570px 宽，远大于视口（660px）

2. **线程错误**（次要）
   - `DisplayInformation.GetForCurrentView()` 在后台线程调用
   - 但这不是导致显示不完整的主因

### 根本原因
- **错误的缩放模型**：用“缩略图的坐标体系”去显示“高清图”
- **尺寸获取失败**：`PixelWidth/PixelHeight` 可能为 0
- **线程问题**：UI API 在后台线程调用

---

## 解决方案

### 核心修复思路

1. **确保高清图尺寸获取**
   - 强制等待图片解码完成
   - 或使用 WIC 解码信息

2. **正确的缩放计算**
   - 每张图重新计算 `fitScale`
   - 按比例换算新的 `zoom`
   - 用中心点锁定避免跳变

3. **线程安全**
   - 在 UI 线程获取显示器分辨率并缓存

### 步骤 3：修复高清图尺寸获取
```csharp
private async Task<(double width, double height, ImageSource source)> GetHighResolutionImageAsync(StorageFile file, uint targetLongSide)
{
    var thumbnailService = App.GetService<IThumbnailService>();
    var imageSource = await thumbnailService.GetThumbnailByLongSideAsync(file, targetLongSide, CancellationToken.None);
    
    // 等待解码完成
    int retry = 0;
    while (retry < 5)
    {
        if (imageSource is BitmapSource bitmap && bitmap.PixelWidth > 0 && bitmap.PixelHeight > 0)
        {
            return (bitmap.PixelWidth, bitmap.PixelHeight, imageSource);
        }
        await Task.Delay(10);
        retry++;
    }
    
    // 回退方案：使用文件信息
    return (_imageFileInfo!.Width, _imageFileInfo.Height, imageSource);
}
```

### 步骤 4：正确计算缩放
```csharp
var (highResWidth, highResHeight, imageSource) = await GetHighResolutionImageAsync(_imageFileInfo.ImageFile, _targetDecodeLongSide);

// 计算新的 fitScale
var vWidth = ImageScrollView.ViewportWidth;
var vHeight = ImageScrollView.ViewportHeight;
double newFitScale = 1.0;
if (vWidth > 0 && vHeight > 0 && highResWidth > 0 && highResHeight > 0)
{
    var scaleX = vWidth / highResWidth;
    var scaleY = vHeight / highResHeight;
    newFitScale = Math.Min(scaleX, scaleY);
    newFitScale = Math.Max(newFitScale, 0.1);
}

// 按比例换算新的 zoom
var currentZoom = ImageScrollView.ZoomFactor;
double newZoom = currentZoom;
if (_thumbnailWidth > 0 && highResWidth > 0)
{
    newZoom = currentZoom * (_thumbnailWidth / highResWidth);
}

// 应用新缩放
ImageScrollView.ZoomTo((float)newZoom, null);
_fitScale = newFitScale;
```

### 步骤 5：中心点锁定
```csharp
// 记录当前中心点
var currentCenterX = ImageScrollView.HorizontalOffset + ImageScrollView.ViewportWidth / 2;
var currentCenterY = ImageScrollView.VerticalOffset + ImageScrollView.ViewportHeight / 2;

// 替换图片...

// 恢复中心点
ImageScrollView.ScrollTo(currentCenterX, currentCenterY);
```
